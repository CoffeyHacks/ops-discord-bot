# OPS-Discord-Bot

## Description

Program allows for deployment of a bot to your desired Discord Server/Channel that allows for searching using Open People Search directly from Discord.

## Installation

1) Create a bot to use with this application in Discord Developer Portal

2) clone this repository to your local machine

3) cd into the directory

4) install "requirements.txt"

5) create .env file with these values:
    Discord_Token
    OPS_USERNAME
    OPS_PASSWORD
    OPS_TOKEN
